## GitHub 업로드 전 1분 보안 체크리스트

### 0) 민감 데이터 정리
- [ ] `python scripts/purge_local_data.py` 실행
- [ ] `data/` 아래 세션/로그/DB 파일이 비워졌는지 확인

### 1) 커밋 제외 규칙 확인
- [ ] `.gitignore`에 `data/`, `*.db`, `*.log`, `.env` 포함 확인
- [ ] API 키/토큰/비밀번호가 코드에 없는지 확인

### 2) 업로드 대상 확인
- [ ] Git 상태에서 커밋 예정 파일 목록 확인
- [ ] 고객 입력/반품사유/대화 원문 파일이 포함되지 않았는지 확인

### 3) 배포 설정 확인
- [ ] 환경변수 방식으로만 비밀값 주입하도록 설정
- [ ] `render.yaml`/배포 스크립트에 비밀값 하드코딩 없음

### 4) 최종 안전 확인
- [ ] 공개 저장소로 올라가도 되는 파일만 남았는지 최종 점검
- [ ] 의심 파일이 있으면 업로드 중단 후 재검토
